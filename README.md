#  Expense Tracker (Python)

##  Project Description
The Expense Tracker is a menu-driven Python application that helps users record and analyze daily expenses.
Users can add expenses under different categories and view both category-wise and total expenses.

This project is suitable for Python beginners and helps build a foundation for data handling and analysis, which is useful for Data Science.

---

##  Features
- Add expenses with category and amount
- View all recorded expenses
- Category-wise expense calculation
- Total expense calculation
- Simple menu-driven interface
- Runs continuously until user exits
